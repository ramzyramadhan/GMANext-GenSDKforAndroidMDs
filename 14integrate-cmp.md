This page covers the comparisons in integrating a consent management platform
(CMP) between Google Mobile Ads SDK (Legacy) and GMA Next-Gen SDK.

## Stop using ad unit deployment

GMA Next-Gen SDK doesn't support
[ad unit deployment](https://support.google.com/admob/answer/14237900). If you decide
to continue using Google's CMP, add the UMP SDK in your app to display your
European regulations message. For more information, see
[Set up UMP SDK](https://developers.google.com/admob/android/next-gen/privacy).
