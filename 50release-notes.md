| 2.1.0 | 2023-07-24 | This release introduces several new APIs to simplify the consent gathering process. Adopting these new APIs is not required for existing integrations. - Calling `https://developers.google.com/admob/android/next-gen/reference/privacy/com/google/android/ump/ConsentInformation#requestConsentInfoUpdate(android.app.Activity,%20com.google.android.ump.ConsentRequestParameters,%20com.google.android.ump.ConsentInformation.OnConsentInfoUpdateSuccessListener,%20com.google.android.ump.ConsentInformation.OnConsentInfoUpdateFailureListener)` **is now required** before interacting with other public APIs. Before calling it, the following are returned: - `https://developers.google.com/admob/android/next-gen/reference/privacy/com/google/android/ump/ConsentInformation.ConsentStatus` returns `https://developers.google.com/admob/android/next-gen/reference/privacy/com/google/android/ump/ConsentInformation.ConsentStatus#UNKNOWN` - `https://developers.google.com/admob/android/next-gen/reference/privacy/com/google/android/ump/ConsentInformation.PrivacyOptionsRequirementStatus` returns `https://developers.google.com/admob/android/next-gen/reference/privacy/com/google/android/ump/ConsentInformation.PrivacyOptionsRequirementStatus#public-static-final-consentinformation.privacyoptionsrequirementstatus-unknown` - `https://developers.google.com/admob/android/next-gen/reference/privacy/com/google/android/ump/ConsentInformation#canRequestAds()` returns `false` - Added a helper method `https://developers.google.com/admob/android/next-gen/reference/privacy/com/google/android/ump/ConsentInformation#public-abstract-boolean-canrequestads`. - Added a static method `https://developers.google.com/admob/android/next-gen/reference/privacy/com/google/android/ump/UserMessagingPlatform#loadAndShowConsentFormIfRequired(android.app.Activity,%20com.google.android.ump.ConsentForm.OnConsentFormDismissedListener)` to combine load and show calls. This method is intended for the use case of showing a form if needed when the app starts. - **Privacy Options** - Added `https://developers.google.com/admob/android/next-gen/reference/privacy/com/google/android/ump/ConsentInformation#getPrivacyOptionsRequirementStatus()` to indicate whether privacy options are required to be shown in this session. - Added a static method `https://developers.google.com/admob/android/next-gen/reference/privacy/com/google/android/ump/UserMessagingPlatform#showPrivacyOptionsForm(android.app.Activity,%20com.google.android.ump.ConsentForm.OnConsentFormDismissedListener)` to be called when users interact with your app's privacy setting. - Enabled hardware acceleration on the consent form's window to improve performance of the consent form in game engines such as Unity. |
